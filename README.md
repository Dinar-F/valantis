## Getting Started

To set up a local copy of the project follow the steps below.

## Installation

1. Clone to local machine

```sh
git clone https://github.com/dinar-f/valantis.git
```

2. Install NPM packages

```sh
npm install
```

3. Start program

```sh
npm run dev
```
